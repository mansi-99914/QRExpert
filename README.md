# QRExpert - Advanced QR Code Generator

A production-ready, highly customizable QR Code Generator built with vanilla JavaScript. It allows users to generate QR codes with custom colors, logos, and error correction levels, featuring a modern glassmorphism UI and offline history support.

## âœ¨ Key Features

* **Real-time Generation:** Instant QR rendering as you type using `EasyQRCodeJS`.
* **Customization:** Pick custom foreground/background colors and adjust size (200px - 600px).
* **Logo Embedding:** Upload brand logos to embed directly into the center of the QR code.
* **Theme System:** Toggle between Light and Dark modes with persistent local storage preference.
* **History Tracking:** Automatically saves the last 5 generated QR codes locally.
* **Export Options:** One-click download (PNG) or Copy to Clipboard.
* **Responsive UI:** Mobile-first design using CSS Grid and Glassmorphism aesthetics.
